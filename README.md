# 🍽️ Restaurant Website

A modern restaurant website built with **HTML5** and **CSS3**.

This project focuses on creating a clean and elegant restaurant landing page using **CSS Flexbox** to build well-structured layouts and improve front-end development skills.

---

## ✨ Features

- 🍽️ Modern Restaurant Landing Page
- 📐 Built with CSS Flexbox
- 🎨 Clean & Elegant User Interface
- 🧩 Semantic HTML5 Structure
- ✨ Clean, Readable & Maintainable Code
